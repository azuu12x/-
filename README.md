# אונבורדינג – ניהול ביטוח נסיעות לחו״ל בשת״פים

אתר למידה אינטראקטיבי בעברית המיועד למנהל פעילות הביטוח נסיעות לחו״ל בשיתוף פעולה (לדוגמה: אלעד).  
האתר סטטי לחלוטין, מבוסס HTML/CSS/Vanilla JS ותוכן ב־`content.json`.

## מבנה הפרויקט

- `index.html` – קובץ ה־HTML הראשי (SPA פשוט).
- `styles.css` – עיצוב מודרני כהה, RTL, מותאם לדסקטופ ומובייל.
- `app.js` – לוגיקה:
  - ניווט בין מסכים (בית, מודולים, תרחישים, קוויז, אנשי מפתח, התקדמות).
  - טעינת תוכן מתוך `content.json`.
  - סימון מודולים כהושלמו ושמירה ב־`localStorage`.
  - הצגת תרחישים אינטראקטיביים.
  - קוויז עם ציון והסברים.
  - הצגת אנשי מפתח עם חיפוש וסינון.
  - מסך התקדמות אישית.
- `content.json` – התוכן עצמו:
  - `modules` – מודולי לימוד.
  - `scenarios` – תרחישים אינטראקטיביים.
  - `quiz` – שאלות קוויז.
  - `contacts` – אנשי קשר פנימיים (ואפשר להרחיב לחיצוניים).
- `README.md` – המסמך הזה.

## איך להריץ מקומית

### אופציה 1 – לפתוח ישירות (פחות מומלץ)

1. שמור את כל הקבצים באותה תיקייה.
2. פתח את `index.html` בדפדפן (double‑click).

> הערה: בדפדפנים מסוימים טעינת `content.json` עם `fetch` עלולה להיחסם בפתיחה ישירה מקובץ (`file://`).  
> אם אתה רואה בעיות בטעינת תוכן, השתמש באופציה 2.
